# Cael Morren — demo vault

This vault is **original fiction**, invented specifically as a demonstration for
Dwell. It is not derived from or set in any existing fictional universe; every
page was written for this project (the `sources: [world-bible]` entries are a
placeholder — there is no external source).

It is a small secondary world (24 pages) built to exercise Dwell's narrative
features: **guided paths** (the Quickening gives every journey a live tension to
move toward), the **creativity dial** (story form + high creativity dreams
scenes from the canon), and **ghost doors** (six lore links — the Ninth
Prophecy, the Third City, and others — deliberately have no pages: they are the
world's unwritten frontier).
