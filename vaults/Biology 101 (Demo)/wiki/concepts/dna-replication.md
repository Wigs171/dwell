# DNA Replication

**DNA replication** is the process by which a cell copies its [[dna]] so that each new cell receives a complete set of instructions. It is described as semiconservative, because when the double helix is copied, each of the two original strands is kept and used as a template for building a new partner strand. The result is two identical double helices, each made of one old strand and one newly assembled strand. This faithful copying must happen before a cell can divide.

Replication begins when an [[enzyme]] called helicase unwinds and separates the two strands, breaking the hydrogen bonds between paired bases and opening a region shaped like a fork. The exposed bases then act as a guide. An enzyme called DNA polymerase moves along each template and adds new nucleotides one at a time, always obeying complementary base pairing so that adenine matches thymine and guanine matches cytosine. In this way the original sequence is preserved with remarkable accuracy, and built-in proofreading corrects most mistakes.

Because DNA polymerase can only add nucleotides in one direction, the two strands are copied differently. One template, the leading strand, is copied smoothly and continuously toward the moving fork. The other, the lagging strand, must be built in short pieces that are later stitched together. Once replication is finished, each [[chromosome]] consists of two identical copies joined together, ready to be separated during [[mitosis]] so that both daughter cells inherit the same genetic information.
